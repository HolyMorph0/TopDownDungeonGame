# TopDown Dungeon Game

A small 2D top-down dungeon game prototype developed with Unity and C#.

## Project Goal

The goal of this project is to improve my game development skills by building a small but complete playable prototype.

## Planned Features

- Top-down player movement
- Camera follow system
- Basic enemy AI
- Health system
- Melee attack system
- Item pickup system
- Game over screen
- Main menu

- Unity
- C#
- Git
- GitHub
